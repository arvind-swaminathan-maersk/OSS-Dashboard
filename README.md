# OSS-Dashboard
SAP OSS Coaching Dashboard
1. Select your workstream: Use the dropdown — dashboard filters to your team's data only
2. Scan the heatmap: Look for the darkest red cells — these are your biggest improvement opportunities
3. Click the red cell: Read the DO / AVOID coaching advice — it tells you precisely what to change
4. Check the Coaching Summary: The bottom section auto-generates your top 2 priorities with specific actions
5. Cross-reference: Use the Error Category and Priority panels to understand the root cause mix.
